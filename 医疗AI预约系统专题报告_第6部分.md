## 第四节、操作程序

### 4.1 新用户入门

#### 4.1.1 注册账号
1. 访问系统首页
2. 点击"注册"按钮
3. 填写注册表单:
   - 真实姓名
   - 联络电话
   - 身份证号
   - 用户名（登录使用）
   - 密码
4. 点击"注册"按钮
5. 看到"注册成功"提示后，点击"登入"

#### 4.1.2 登录系统
1. 输入用户名和密码
2. 点击"登入"按钮
3. 成功后自动跳转到AI助手主页

### 4.2 使用AI助手

#### 4.2.1 文字对话
1. 在聊天界面底部的输入框输入问题
2. 点击发送按钮（纸飞机图标）或按Enter键
3. 等待AI回复（通常2-4秒）
4. 查看AI的回答

**示例对话**:
- "感冒有哪些症状？"
- "高血压患者饮食要注意什么？"
- "头痛怎么办？"

#### 4.2.2 语音输入
1. 点击麦克风按钮（允许浏览器使用麦克风）
2. 看到"正在录音中..."提示
3. 对着麦克风说话
4. 再次点击麦克风按钮停止录音
5. 系统自动转换为文字并发送给AI
6. 查看AI回复

**语音输入技巧**:
- 在安静环境下录音
- 说话清晰、速度适中
- 避免长时间录音（建议30秒内）

#### 4.2.3 快速范例
点击界面上方的范例按钮，系统会自动发送预设问题：
- 预约门诊范例
- 常见疾病资讯

### 4.3 预约门诊

#### 4.3.1 方式一：使用预约表单
1. 点击右上角"预约门诊"按钮
2. 填写预约信息:
   - 病历号（选填）
   - 科别（下拉选择）
   - 医师姓名（根据科别自动更新）
   - 预约日期（最早明天）
   - 预约时间（默认09:00）
   - 症状描述（选填）
3. 姓名和电话会自动从个人资料填入
4. 检查信息无误后点击"提交预约"
5. 成功后自动跳转到"我的预约"页面

#### 4.3.2 方式二：AI智能预约
1. 在聊天框输入预约意愿，例如:
   - "我要预约内科"
   - "帮我预约明天的眼科"
   - "预约明天上午10点的骨科，医师是吴骨晨"

2. AI会自动识别并提取信息
3. 如信息不完整，AI会询问缺失的内容
4. 逐步补充信息
5. 信息完整后，系统自动创建预约

**完整示例对话**:
```
用户: "我要预约内科"
AI: "好的，请告诉我预约日期和时间"
用户: "明天上午10点"
AI: "请问要预约哪位医师？"
用户: "张内晨医师"
AI: "预约已成功创建！"
```

### 4.4 管理预约

#### 4.4.1 查看预约记录
1. 点击右上角"我的预约"按钮
2. 查看所有预约记录
3. 每条记录显示:
   - 病历号、姓名、电话
   - 科别、医师
   - 预约日期、时间
   - 症状描述
   - 预约状态

#### 4.4.2 修改预约
1. 在预约列表找到要修改的预约
2. 点击"编辑"按钮
3. 修改需要更改的字段
4. 点击"提交"保存修改

#### 4.4.3 取消预约
1. 在预约列表找到要取消的预约
2. 点击"取消预约"按钮
3. 确认取消
4. 预约状态变为"已取消"

### 4.5 个人资料管理

1. 点击右上角用户名
2. 查看个人资料:
   - 姓名、电话（可修改）
   - 身份证号（部分遮蔽显示，可修改）

3. 修改需要更新的字段
4. 点击"更新资料"保存

### 4.6 长辈模式使用

#### 4.6.1 启用长辈模式
1. 点击右上角"长辈模式"按钮
2. 界面切换为简化版:
   - 隐藏复杂元素
   - 显示超大麦克风按钮
   - 大字体状态提示

#### 4.6.2 使用长辈模式

### 5.2 常见问题处理

#### 5.2.1 AI无法回复
**问题**: AI聊天没有响应  
**解决方法**:
1. 检查`.env`文件中的`GEMINI_API_KEY`是否正确
2. 确认API配额未用完
3. 检查网络连接

#### 5.2.2 语音识别失败
**问题**: 语音转文字不准确  
**解决方法**:
1. 确保在安静环境下录音
2. 说话清晰、速度适中
3. 检查麦克风权限是否已授予
4. 尝试使用不同浏览器

**问题**: 预约无法创建/查询  
**解决方法**:
1. 检查数据库文件是否存在
2. 重新运行`init_sqlite_db.py`
3. 检查文件权限

**问题**: 访问localhost:5000无响应  
**解决方法**:
1. 确认Flask应用正在运行
2. 检查端口5000是否被占用
3. 查看控制台错误信息
4. 重启应用

4. **更新系统**: 及时更新Python和依赖包
